# Modular JavaScript Framework for Reactive and Asynchronous Data Processing

## Overview

This project demonstrates a modular JavaScript framework for asynchronous and reactive data processing.

The system combines multiple architectural approaches and design patterns into one project.

## Features

- Event-driven communication
- Memoization and caching
- Async data processing
- Authentication proxy
- Logging decorators
- Dependency Injection

## Design Patterns

- Proxy
- Decorator
- Observer/EventBus
- Dependency Injection

## Project Structure

```text
library/   -> core modules
example/   -> demo examples
```

## Run Example

```bash
node example/appDemo.js
```

## Technologies

- JavaScript (ES Modules)
- Node.js
